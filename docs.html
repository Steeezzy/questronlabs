API Introduction

Welcome to the Qestron Labs Developer Documentation. Our REST API allows you to trigger full autonomous software sprints, connect into our agent chatstreams, or provision continuous background builds securely with simple HTTP endpoints.

All response objects return clean structured application handles safely for client consumption correctly. Standard JSON responses provided consistently.

Authentication

Authenticate requests by passing your live token inside the standard header setup properly safely.

Autonomous Projects Sprints

The main trigger to spin up fully parallel simulated work environments instantly.

POST /v1/project

Trigger a parallel AI society model run for building complete code setups correctly.

FieldTypeNotes
titlestringFull name of Sprint build target structure
descstringScope details regarding setup features needed
agentsintegerQuantity of simulate triggers (Max 10,000)
